# 真実の知識データベース

このドキュメントでは、占いアプリに追加された「真実の知識データベース」機能について説明します。

## 📖 概要

真実の知識データベースは、九星気学や陰陽五行思想の**本質的な理由**を収集・管理するためのシステムです。

「なぜそうなのか？」という深い理解を提供することで、表面的な占いではなく、宇宙の原理に基づいた本質的な知識を伝えることができます。

## 🎯 主な特徴

### 1. 本質的な「なぜ？」を含む説明

従来の占いアプリとは異なり、以下のような深い情報を提供します：

- **宇宙的原理**: なぜその星がその性質を持つのか（惑星の動き、洛書の配置など）
- **五行的理由**: なぜその五行と結びつくのか（自然の法則、相生相剋の原理）
- **歴史的背景**: どのように発展してきたか（中国から日本への伝来、時代による変化）
- **失われた知識**: 明治維新やGHQ占領期に失われた/歪められた本来の知識

### 2. 開発者が随時追加できる仕組み

- Supabaseデータベースに知識を保存
- 開発者用管理画面（`/admin/knowledge`）でいつでも閲覧・追加可能
- カテゴリ分類とタグ付けで整理

### 3. 検証済みの情報源

収集した情報には以下のソースを含みます：

- [陰陽五行思想 - Wikipedia](https://ja.wikipedia.org/wiki/陰陽五行思想)
- [九星気学 - Wikipedia](https://ja.wikipedia.org/wiki/九星気学)
- [明治改暦 - Wikipedia](https://ja.wikipedia.org/wiki/明治改暦)
- その他、歴史的文献と伝統的教え

## 🗄️ データベース構造

### テーブル一覧

1. **knowledge_categories** - 知識カテゴリ
   - 陰陽五行思想
   - 九星気学の原理
   - 日本の伝統的暦
   - 失われた知識
   - 宇宙と人間の関係
   - 方位学の本質
   - 風水の真理
   - 歴史的背景

2. **knowledge_entries** - 知識エントリ
   - title: タイトル
   - content: 本文
   - summary: 要約
   - **essence**: 本質（なぜそうなのか）
   - **historical_context**: 歴史的背景
   - **traditional_wisdom**: 伝統的な教え
   - **lost_knowledge**: 失われた/歪められた知識
   - importance_level: 重要度（1-5）

3. **knowledge_tags** - タグ
   - 本質、歴史、真実、宇宙原理、陰陽五行、失われた知識、伝統、科学的根拠

4. **five_elements** - 五行の詳細データ
   - 木・火・土・金・水それぞれの本質的性質
   - 宇宙的原理、自然界での現れ、人間界での現れ
   - 相生相剋の関係

## 🚀 セットアップ手順

### 1. Supabaseマイグレーションの実行

Supabaseダッシュボードの**SQL Editor**で以下のファイルを順番に実行してください：

```sql
-- 1. 知識データベースのスキーマ作成
supabase/migrations/20260102000000_knowledge_database.sql

-- 2. 初期データの投入
supabase/migrations/20260102000001_initial_knowledge_data.sql
```

### 2. 開発者用管理画面へのアクセス

ブラウザで以下のURLにアクセス：

```
http://localhost:3000/admin/knowledge
```

### 3. 知識の追加

管理画面から新しい知識エントリを作成できます（現在はダミーデータ表示中）。

実装時には、Supabaseクライアントを使ってデータベースと接続してください。

## 💡 活用方法

### 既存の知識ベースへの統合

`lib/fortune/nine-star-ki/knowledge-base.ts`に以下のフィールドを追加：

```typescript
essence: {
  core: string;              // 核心的性質
  cosmicPrinciple: string;   // 宇宙的原理（なぜその性質を持つのか）
  elementalReason: string;   // 五行的理由（なぜその五行なのか）
  lifeDirection: string;     // 人生の方向性
  innerNature: string;       // 内面の本質
  spiritualPath: string;     // 精神的な道
}
```

これにより、占いの説明に自然に「なぜそうなのか」が織り込まれます。

### 例: 一白水星の説明

**従来**:
> 「一白水星の人は柔軟で深い洞察力を持ちます」

**改良後**:
> 「一白水星は洛書の配置で「1」の位置、北方に配されます。北は陰が極まり、新たな陽が胎動する場所。水星（惑星）は約88日という最速の公転周期を持ち、その俊敏な動きから知恵と流動性の象徴とされました。水は形を持たず器に従いますが、集まれば大海となり、凍れば岩よりも硬くなります。この柔軟性と潜在的な強大さが、一白水星の人の深い洞察力と本質を見抜く力として現れるのです」

## 🔍 失われた知識の例

### 明治改暦の真実

多くの人は「明治時代に西洋化のために暦が変わった」と思っていますが、真実は：

- **財政難が主な理由**: 明治5年12月を2日間だけにし、翌日を明治6年1月1日とすることで、2ヶ月分の給料支払いを免れた
- **わずか3週間前の告知**: 1872年11月9日に発表、12月3日実施。地方には情報が届かず大混乱
- **暦注の禁止**: 大安・仏滅などの吉凶情報を「迷信」として完全禁止
- **「お化け暦」の流通**: 民衆の反発により、密かに暦注入りの非公式暦が流通
- **戦後まで続いた旧暦**: 多くの農村部では1950〜60年代まで旧暦を使用

### GHQではなく明治政府

一般的に「GHQが占いを禁止した」と思われていますが、実際は：

- 暦注の禁止は**明治政府**による（1872年）
- GHQ占領期（1945-1952）には、むしろ**1946年に暦の出版が自由化**された
- 占いが「迷信」として否定されたのは明治の西洋化政策

## 🔐 セキュリティ注意事項

**重要**: `/admin/knowledge`ページは現在認証なしでアクセス可能です。

本番環境にデプロイする前に：

1. Supabase Authを使った認証を実装
2. Row Level Security (RLS)ポリシーで書き込みを制限
3. 環境変数で管理者のメールアドレスを設定
4. 認証されたユーザーのみがアクセスできるように保護
