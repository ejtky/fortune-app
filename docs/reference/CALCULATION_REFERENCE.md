# 暦計算・九星気学 参照資料

## 参照サイト
- **名称**: 暦計算サイト
- **URL**: https://keisan.site/menu/system/000000000220

## ライブラリ・計算機能一覧

| ライブラリ名 | 概要 |
| :--- | :--- |
| **旧暦カレンダー** | 現行年月から旧暦カレンダーを計算します。 |
| **旧正月（春節）はいつ？** | 旧暦正月（春節）を現行年月に変換します。 |
| **六曜計算** | 現行暦日から旧暦に変換し六曜を計算します。 |
| **干支カレンダー** | 現行年月から干支カレンダー（日の干支、月の干支、年の干支）を計算します。 |
| **干支計算** | 西暦または和暦の年から干支（エト）を計算します。 |
| **九星カレンダー** | 現行年月から九星カレンダー（日家九星、月家九星、年家九星）を計算します。 |
| **九星盤計算** | 指定日の年家九星、月家九星、日家九星を求め、それぞれの九星盤が表示されます。 |
| **時の九星** | 現行年月日と時刻から年月日家九星と時の九星（時家九星）を計算します。 |
| **暦注カレンダー** | 現行年月から旧暦カレンダーと暦注（干支、九星、十二直、二十八宿/二十七宿）を計算します。 |
| **暦注計算** | 現行暦日の暦注（旧暦、六曜、干支、九星、十二直、二十八宿、二十七宿）を計算します。 |
| **二十四節気** | 一年間の二十四節気（小寒、大寒、立春、...）を計算します。 |
| **雑節** | 一年間の雑節（土用、節分、彼岸、社日、八十八夜、入梅、半夏生、二百十日）を計算します。 |
| **週カレンダー** | 指定都市と年月日から1週間分のカレンダー（六曜、九星、日の出、日の入、月齢）を計算します。 |
| **恵方の方角の計算** | 西暦から恵方（えほう)の方角を計算します。歳徳神の在する方位を恵方と言い、陰陽道では福徳の神とされ、その方角に向かって事を行えば縁起が良いとされています。 |
| **五節句** | 五節句（人日、上巳、端午、七夕、重陽）を計算します。 |
